# ADR-005: SQLite canonical store with docs derived

Status: Accepted
Date: 2026-03-30

Decision:
- SQLite is source of truth.
- `docs/` artifacts are derived and rebuildable.
- `rebuild-docs` must not mutate canonical rows.
