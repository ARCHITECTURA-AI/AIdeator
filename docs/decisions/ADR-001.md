# ADR-001: Local-first, tiered runtime modes

Status: Accepted
Date: 2026-03-30

Decision:
- `local-only`: no external calls.
- `hybrid`: keyword-only external search; local LLM.
- `cloud-enabled`: full context to configured cloud providers.
- Mode is immutable for a run.
