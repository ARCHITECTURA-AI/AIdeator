# ADR-002: Async run model with polling

Status: Accepted
Date: 2026-03-30

Decision:
- `POST /runs` returns 202 quickly.
- Client polls `GET /runs/{id}/status` until terminal.
- Avoids long-lived HTTP blocking and keeps MVP infra simple.
